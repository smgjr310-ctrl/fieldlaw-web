# FieldLaw Web

FieldLaw Web is a mobile-first Next.js + TypeScript version of the FieldLaw field reference demo.

## Constraints

- No AI
- No login
- No backend
- No external APIs
- Local static data only
- Vercel deployable

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm start
```

## Deploy to Vercel

Import this folder as a Vercel project. The default Next.js settings work.

## Content Included

- Utah State Code/common charge references
- Utah code sections
- Charge elements
- Officer checklists
- Evidence checklists
- Enhancement checklists
- Related charges
- Search warrant guides
- Report guides
- DUI/SFST checklist
- About and disclaimer

All legal content is demo reference content and must be verified with current Utah Code and agency policy.
